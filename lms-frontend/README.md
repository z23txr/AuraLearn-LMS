# AuraLearn - Modern MERN Stack LMS

AuraLearn is a full-stack Learning Management System (LMS) designed for seamless interaction between instructors and students.

## 🚀 Features
- **Secure Authentication**: JWT-based login and registration with role-based access.
- **Password Recovery**: Token-based email recovery system using Nodemailer.
- **Strong Security**: Password validation requiring uppercase, lowercase, numbers, and symbols.
- **Modern UI**: Interactive frontend built with React, Vite, and Framer Motion animations.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Axios, React-Toastify.
- **Backend**: Node.js, Express.js, MongoDB (Mongoose).
- **Automation/Tools**: Nodemailer for emails, Thunder Client for API testing.

## 🔧 Installation
1. Clone the repo: `git clone https://github.com/z23txr/AuraLearn-LMS.git`
2. Setup `.env` in the backend folder with your `MONGO_URI`, `JWT_SECRET`, and Gmail App Password.
3. Run `npm install` in both frontend and backend folders.
